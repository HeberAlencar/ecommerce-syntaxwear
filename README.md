# SyntaxWear 👟

## 📖 Sobre o Projeto
O **SyntaxWear** é uma interface de e-commerce moderna e responsiva focada na venda de tênis e sneakers. O projeto foi desenvolvido como um protótipo de alta fidelidade para demonstrar habilidades de design de interface (UI) e desenvolvimento front-end utilizando tecnologias web fundamentais.

A interface foca na experiência do usuário (UX), com uma estética urbana e futurista que destaca os produtos de forma clean e impactante.

## 🚀 Funcionalidades
- **Navegação Responsiva:** Cabeçalho com menu mobile (hamburguer) funcional via CSS e acesso rápido a categorias e conta do usuário.
- **Seção Hero de Impacto:** Chamada para ação (CTA) para o modelo principal "Krypton One".
- **Navegação por Categorias:** Cards visuais para os estilos: Casual, Esporte, Moderno e Futurista.
- **Grid de Produtos:** Layout flexível e organizado para exibição da coleção de sneakers.
- **Newsletter:** Área de captura de e-mails para marketing integrada ao rodapé.
- **Rodapé Institucional:** Links de navegação detalhados e ícones de redes sociais.

## 🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:
- **HTML5:** Estruturação semântica para melhor acessibilidade e SEO.
- **CSS3 (Moderno):**
  - **CSS Variables:** Para fácil manutenção de temas, cores e tipografia.
  - **Flexbox & CSS Grid:** Para criação de layouts complexos e responsivos.
  - **Arquitetura de Componentes:** Estilos divididos por arquivos (ex: `header.css`, `hero.css`) para facilitar a manutenção.

## 📁 Estrutura do Projeto
```text
ecommerce-syntaxwear/
├── css/
│   ├── base.css            # Estilos base de tags e tipografia
│   ├── layout.css          # Estrutura principal das seções
│   ├── reset.css           # Reset de estilos padrão dos navegadores
│   ├── variables.css       # Definição de cores, fontes e tamanhos
│   └── components/         # Estilos modulares de cada parte do site
├── images/
│   ├── banners/            # Imagens de fundo e hero
│   ├── icons/              # Ícones SVG (bag, user, social, etc.)
│   ├── logo/               # Identidade visual da marca
│   └── products/           # Fotos dos modelos de tênis
├── js/                     # Scripts de interatividade (em desenvolvimento)
└── index.html              # Arquivo principal do site
```

## 💻 Como Visualizar
1. Faça o download ou clone este repositório.
2. Localize o arquivo `index.html` na pasta raiz.
3. Abra o arquivo em qualquer navegador web moderno (Chrome, Firefox, Edge, etc.).
